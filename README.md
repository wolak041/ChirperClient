## Requirements
* [Node](https://nodejs.org/)

## Available scripts
### Run app
```sh
npm run start
```

### Build app
```sh
npm run build
```

### Build app and copy it to public folder in api
```sh
npm run api:build
```

### Clear public folder in api
```sh
npm run api:clear-build
```

### Copy build to public folder in api
```sh
npm run api:copy-build
```

### Run prettier watch
```sh
npm run prettier-watch
```

### Run eslint
```sh
npm run lint
```
